Detailed Description

Base class for SX130x series LoRaWAN concentrators. All derived classes for SX130x (e.g. SX1301 or SX1302) inherit from this base class. This class should not be instantiated directly from user code, only from its derived classes.

Unlike single-channel transceivers, the SX130x is a multi-channel concentrator designed for LoRaWAN gateway applications. It can receive on 8 multi-SF LoRa channels simultaneously, plus 1 LoRa single-SF channel and 1 FSK channel.
